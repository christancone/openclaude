Build an audit-grade graph

Either paste a Sparengine asset id (the OCR pipeline fetches the CSV from Supabase) or upload a CSV directly. The agent reads the dossier rows, builds the knowledge graph in Neo4j, runs the QA layers, and redirects you to the Neo4j Browser pre-filled with a query scoped to that asset.

Comma- or newline-separated. The first id is used for the workdir name when multiple are submitted.
